# Agent

## 现代agent架构

| 架构类别          | 决策风格              | 典型应用场景              | 架构特点                          | 优点                          | 常见实现                  |
|-------------------|-----------------------|---------------------------|-----------------------------------|-------------------------------|---------------------------|
| ReAct / 工具调用 | 推理-行动紧密交替    | 知识问答、自动化任务     | 推理和行动交替进行，使用工具调用 | 实时反馈，提高准确性         | ReAct框架，LangChain工具调用 |
| Reflexion / 反思型 | 事后总结，自我修正  | 代码生成、数学推理       | 执行后反思，自我修正             | 学习和改进性能                | Reflexion论文方法         |
| Plan-Execute     | 先计划后执行         | 复杂多步问题、机器人任务 | 先规划，然后执行计划             | 结构化，适合复杂任务          | Plan-and-Execute模式     |
| 多 Agent 协作    | 分工、辩论、角色扮演 | 软件开发、内容生产、复杂模拟 | 多个agent分工合作               | 并行处理，模拟人类团队        | Multi-Agent Systems，如CrewAI |
| CoALA 认知架构   | 模块化认知循环       | 长期自主 Agent 设计蓝图  | 模块化认知循环                   | 灵活，适用于长期任务          | CoALA框架                |
| AutoGPT 自主任务拆解 | 自动生成与调度子任务 | 开放式目标实现、研究助手 | 自动拆解任务并调度               | 自主性强                      | AutoGPT项目              |
